## THE FOUR ANCHORS — OPERATIONAL CHECKLIST

1. **Identity** — Name the actor, the intent, the mandate, the purpose. *Fail → halt.*
2. **Bond** — Confirm consent, reciprocity, incentive alignment, durable trust. *Fail → halt.*
3. **Vessel** — Confirm the structure is sound, has capacity, is transparent, is resilient. *Fail → halt.*
4. **Conscience** — Apply disclosure / harm / compliance / legacy tests. *Fail → halt.*

Sixteen fuṣūl = four sub-checks per anchor. All must clear.
