# How to Use This Repository (No Git Knowledge Required)

This guide is written for you, Afra. You don't need to learn Git commands. Everything can be done from the GitHub website.

---

## Viewing Files
1. Go to [your repo on GitHub](https://github.com/alzadjaliaafra-hash/SRFF-I-RVS-Model)
2. Click any folder or file to view it
3. Excel files can be previewed directly on GitHub

## Downloading Files
1. Navigate to the file you want
2. Click the **Download** button (down arrow icon)
3. The file saves to your computer

## Uploading New Files
1. Navigate to the folder where you want to add a file
2. Click **Add file** → **Upload files**
3. Drag and drop your file
4. Click **Commit changes**

## Editing Text Files (like templates)
1. Click on the file
2. Click the **pencil icon** (Edit)
3. Make your changes
4. Click **Commit changes**

## Running the Weekly Health Check Manually
1. Go to the **Actions** tab at the top
2. Click **Weekly RVS Model Health Check** on the left
3. Click **Run workflow** → **Run workflow**
4. Wait 2-3 minutes for results

## Using GitHub Codespaces (Free Dev Environment)
1. Click the green **Code** button on the repo page
2. Click **Codespaces** tab
3. Click **Create codespace on master**
4. A full VS Code editor opens in your browser — run Python, edit files, test models

## Enabling GitHub Copilot (Free AI Assistant)
1. Go to [github.com/features/copilot](https://github.com/features/copilot)
2. Sign up for the free plan
3. Install the Copilot extension in VS Code or use it in Codespaces
4. It will auto-suggest code as you type
